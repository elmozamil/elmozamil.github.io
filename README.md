# Elmozamil Elamir Executive Portfolio V3

A professional GitHub Pages portfolio focused on data and information management leadership, AI strategy, analytics, governance, and consulting services.

## Deploy on GitHub Pages

1. Create a repository named `YOUR_USERNAME.github.io`
2. Upload these files to the repository root:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Go to Settings > Pages
4. Select Deploy from branch > main > root
5. Open `https://YOUR_USERNAME.github.io/`

## Notes

- The Projects section is intentionally left as a professional placeholder.
- Update the GitHub link in `index.html` if your username is different from `elmozamil`.
